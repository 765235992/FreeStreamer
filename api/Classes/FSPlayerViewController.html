Inherits from UIViewController
Conforms to FSAudioControllerDelegate
Declared in FSPlayerViewController.h

Overview

The player view controller of the iOS example application.

The view allows the user to control the player. See the play:, pause: and seek: actions.

shouldStartPlaying

If this property is set to true, the stream playback is started when the view appears. When the playback starts, the property is automatically set to false. For consequent playback requests, the flag must be activated again.

@property (nonatomic, assign) BOOL shouldStartPlaying

Discussion

If this property is set to true, the stream playback is started when the view appears. When the playback starts, the property is automatically set to false. For consequent playback requests, the flag must be activated again.
